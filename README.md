# Bienvenue chez Sirius School

Tu te prépares à embarquer pour 5 mois de plaisir et de (légères) souffrances dans le monde du développement web ! 😈 🔥

Ensemble nous apprendrons les bases des 3 langages principaux du web, autrement dit l'HTML, le CSS et le JavaScript ! 💪 🛠️

> Cours rédigé pour l'année 2023
>
> Auteur: Jeremy Scala & Lucas Ielli

## Au programme

Cette formation s'articule autour d'un tronc commun comprenant les bases des trois langages. Pendant 6-8 semaines tu suivras des cours avec Jeremy en HTML/CSS et avec Lucas en JavaScript.

Après avoir finis cette entrée en matière tu auras l'occasion de t'entretenir avec tes deux formateurs pour décider de quelle partie tu préfères.
