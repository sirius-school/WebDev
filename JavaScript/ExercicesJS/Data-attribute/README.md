# Exercice sur les attributs HTML personnalisés (`data-*`):

## Objectif de l'exercice :
Dans cet exercice, vous devez créer une page HTML avec des attributs `data-*`, puis accéder et modifier ces attributs en utilisant JavaScript.

1. Créez un fichier HTML appelé "index.html".

2. À l'intérieur de "index.html", créez une balise `<div>` avec une `id` unique, par exemple "myDiv". À l'intérieur de cette balise, ajoutez du texte de votre choix.

3. Ajoutez deux attributs `data-*` personnalisés à cette balise :
   - `data-category` avec la valeur "informatique"
   - `data-price` avec la valeur "50"

4. En utilisant JavaScript, sélectionnez l'élément avec l'identifiant "myDiv" et stockez-le dans une variable.

5. Utilisez la propriété `dataset` pour extraire la valeur de l'attribut `data-category` de cet élément et affichez-la dans la console.

6. Utilisez la propriété `dataset` pour extraire la valeur de l'attribut `data-price`, convertissez-la en nombre (car elle est actuellement une chaîne de caractères), puis affichez-la dans la console.

7. Modifiez la valeur de l'attribut `data-price` pour qu'elle devienne "60" en utilisant JavaScript.

8. Affichez la nouvelle valeur de l'attribut `data-price` dans la console pour vérifier qu'elle a été modifiée avec succès.

**Bonus :**

9. Utilisez les valeurs extraites de l'attribut `data-category` et de l'attribut `data-price` pour afficher un message dans la page HTML, par exemple : "Le produit de catégorie [valeur de data-category] coûte [valeur de data-price] euros."

10. Ajoutez du CSS pour appliquer un style particulier à l'élément avec l'attribut `data-category` égal à "informatique".

11. Créez un autre élément HTML avec un attribut `data-*` personnalisé de votre choix, puis accédez et affichez sa valeur à l'aide de JavaScript.
