# Exercice : Manipulation de tableaux

## Objectif de l'exercice :

Dans cet exercice, vous allez pratiquer l'utilisation de différentes méthodes pour manipuler des tableaux en JavaScript. Vous utiliserez les méthodes `isArray`, `fill`, `find`, et `join`.

1. **`isArray`**

    Vous avez un tableau `myArray` qui contient des nombres. Écrivez un code qui vérifie si `myArray` est un tableau ou non en utilisant la méthode `isArray`. Si `myArray` est un tableau, affichez "C'est un tableau !" dans la console, sinon, affichez "Ce n'est pas un tableau !" dans la console.

2. **`fill`**

    Vous avez un tableau `myArray` initialisé avec 5 éléments, tous égaux à 0. Utilisez la méthode `fill` pour remplacer tous les éléments de `myArray` par la valeur `7`. Ensuite, affichez le tableau modifié dans la console.

3. **`find`**

    Vous avez un tableau `numbers` contenant une liste de nombres. Utilisez la méthode `find` pour trouver le premier nombre supérieur à `30` dans le tableau `numbers`. Affichez ce nombre dans la console.

4. **`join`**

    Vous avez un tableau `words` contenant des mots. Utilisez la méthode `join` pour concaténer tous les mots en une seule chaîne de caractères. Utilisez un espace comme séparateur entre les mots. Affichez la chaîne de caractères résultante dans la console.

**Conseils** :
- Assurez-vous de comprendre comment chaque méthode fonctionne en vous référant aux explications du cours.
