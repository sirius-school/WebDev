# Rappel

```js
const // Permet de déclarer une variable constante mais n'est pas réassignable.
```
```js
let // Permet de déclarer une variable qui est réassignable.
```
```js
const firstname = "Lucas"; // Variable de type String
```
```js
let age = 50; // Variable de type Number
```
```js
let isConnected = true; // Variable de type Boolean
```
```js
const courseArray = []; // Variable de type Array 
```
Attention que la variable `courseArray` est elle de type tableau MAIS les données reprisent à l'intérieur auront leur propre type.
```js
const courseObject = {}; // Variable de type Object
```
