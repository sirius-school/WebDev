# Rappel des variables

```js
const // Permet de déclarer une variable constante mais n'est pas réassignable.
```
```js
let // Permet de déclarer une variable qui est réassignable.
```
```js
const hello = "Hello world !"; // Variable de type String
```
```js
let age = 50; // Variable de type Number
```
```js
let isConnected = true; // Variable de type Boolean
let isConnected = false; // Variable de type Boolean
```
```js
const courseArray = ["Valeur", 1]; // Variable de type Array, valeur de type String, deuxième valeur de type Number 
```
> Attention que la variable `courseArray` est elle de type tableau **MAIS** les données reprisent à l'intérieur auront leur propre type.
```js
const courseObject = {
    course: "JavaScript", // Valeur de la propriété de type String
    numberOfLanguage: 3, // Valeur de la propriété de type Number 
}; // Variable de type Object
```
> Attention que la variable `courseObject` est elle de type objet **MAIS** les données reprisent à l'intérieur des propriétés auront leur propre type.
```js
const courseUndefined = undefined; // Variable de type undefined
```
```js
const courseNull = null; // Variable de type null
```
